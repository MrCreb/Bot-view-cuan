# Download and install nvm:
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash
```
# Download and install Node.js:
```bash

nvm install 22
```

```bash

node -v # Should print "v22.13.1".
nvm current # Should print "v22.13.1".
```
